# Python Docstring 작성 가이드

Python에서 클래스, 함수, 모듈 등의 이름 위에 마우스 커서를 올렸을 때(Hovering) 설명 툴팁이 나타나도록 만드는 **Docstring(독스트링) 작성법 및 규칙**을 설명합니다.


---


## **1. Docstring이란?**

* **정의**: 파이썬 코드 내에서 모듈, 클래스, 메서드, 함수 등의 역할을 설명하기 위해 선언문 바로 아래에 작성하는 **문자열 주석**입니다.
* **동작 원리**:
  * 파이썬 컴파일러는 이 문자열을 객체의 특수 속성인 `__doc__`에 자동으로 할당합니다.
  * VS Code, PyCharm 등 현대적인 IDE는 이 `__doc__` 속성을 읽어 마우스 호버(Hover) 시 툴팁 인터페이스로 보기 좋게 렌더링해 줍니다.
* **일반 주석(****`#`****)과의 차이**:
  * 일반 주석(`#`)은 단순 코드 주석으로 컴파일 시 무시되며 호버 툴팁에 표시되지 않습니다.
  * 반면 Docstring(`"""`)은 실제 객체의 메타데이터로 등록되어 호버 툴팁과 자동완성 도움말에 표시됩니다.

---


## **2. Docstring 작성 규칙**


### **① 위치**

클래스, 함수, 모듈 선언 코드 **바로 다음 첫 번째 줄**에 위치해야 합니다. 중간에 공백이나 일반 주석(`#`)이 먼저 오면 Docstring으로 인식되지 않습니다.


```python
# (O) 올바른 위치
class Base:
    """
    여기에 적는 내용이 툴팁으로 보입니다.
    """
    pass

# (X) 잘못된 위치 (중간에 일반 주석이 들어감)
class Base:
    # 이 주석 때문에 아래 문자열은 Docstring이 아닌 일반 문자열이 됩니다.
    """
    이렇게 하면 툴팁에 나타나지 않습니다.
    """
    pass
```


### **② 기호**

* 반드시 **세 개의 큰따옴표(****`"""`****)** 또는 세 개의 작은따옴표(`'''`)로 감싸야 합니다. (파이썬 공식 가이드인 PEP 257에서는 세 개의 큰따옴표 `"""` 사용을 권장합니다.)

### **③ 마크다운(Markdown) 지원**

* VS Code를 비롯한 최신 IDE들은 툴팁 렌더링 시 **마크다운 서식을 지원**합니다.
* 따라서 Docstring 내부에 `항목`, `항목`과 같은 리스트 형태, `*굵게**`, ``코드`` 등의 문법을 사용하면 툴팁 화면에서도 스타일이 적용되어 깔끔하게 보입니다.

## **3. 대표적인 Docstring 스타일 예시**

파이썬 커뮤니티에서는 협업 시 가독성을 높이기 위해 규격화된 스타일 형식을 사용합니다.


### **① Google 스타일 (추천)**

가장 널리 쓰이며 가독성이 높고 직관적입니다.


```python
def calculate_area(width: float, height: float) -> float:
    """사각형의 넓이를 계산합니다.

    두 수(가로, 세로)를 인자로 받아 곱한 결과를 반환합니다.

    Args:
        width (float): 사각형의 가로 길이
        height (float): 사각형의 세로 길이

    Returns:
        float: 계산된 사각형의 넓이

    Raises:
        ValueError: 가로나 세로가 0 이하일 때 발생
    """
    if width <= 0 or height <= 0:
        raise ValueError("길이는 0보다 커야 합니다.")
    return width * height
```


### **② Sphinx / ReStructuredText (RST) 스타일**

파이썬 공식 문서 자동 생성 도구인 Sphinx에서 사용하는 표준 스타일입니다.


```python
def calculate_area(width, height):
    """사각형의 넓이를 계산합니다.

    :param width: 사각형의 가로 길이
    :type width: float
    :param height: 사각형의 세로 길이
    :type height: float
    :returns: 계산된 사각형의 넓이
    :rtype: float
    """
    return width * height
```

## **4. 요약 정리**

| 항목 | 일반 주석 (`#`) | Docstring (`"""`) |
| --- | --- | --- |
| **목적** | 코드 내부 동작에 대한 설명 (개발자용) | 코드 사용자(호출자)를 위한 설명 (인터페이스용) |
| **위치** | 코드 어디나 가능 | 클래스, 함수, 모듈 선언부 바로 아래 첫 줄 |
| **동작** | 컴파일 시 무시됨 | 객체의 `__doc__` 속성으로 빌드됨 |
| **IDE 툴팁 노출** | 노출 안 됨 | **커서 호버 시 툴팁으로 노출됨** |
| **마크다운 지원** | 해당 없음 | **지원함 (리스트, 굵게 등 서식 사용 가능)** |
