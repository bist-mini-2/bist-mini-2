# 📅 3인 전담제 기반 2주 단기 집중 개발 일정 및 WBS (Development Roadmap & WBS - 3rd Milestone)

본 문서는 **6월 30일 프로젝트 완료 및 최종 배포**를 목표로, 3명의 팀원이 **1단계에서 3대 RAG 파이프라인을 공동 구축한 후, 2단계에서 기능 A, B, C, D 중 담당한 핵심 기능을 7일 동안 풀(Full)로 전담하여 구현**한 WBS 일정 계획 및 최종 3차 진행 상태 보고서입니다.

*   **승현**: 생명공학 RAG 파이프라인 (`q-bio.GN` 및 `q-bio.BM/MN/TO/CB/SC/OT` 총 21,383건 추출 가공, `bio_embeddings` 컬렉션 적재, RAG 유사도 검색 API)

### 🛠️ 1.2 [2단계] 기능별 전담 개발 (6/18 ~ 6/26, 7일) - **[완료 및 마감 진행 중]**
*   **Case A: 일반 챗 허브 (General Chat Hub) [완료]**: Step-back 대안 쿼리 생성 에이전트, CoT 추론 SSE 스트리밍, Pydantic Structured Output 인용 출처 매핑, D3.js 기반 인용 관계망 시각화 개발.
*   **Case B: 대규모 문헌 비교 분석기 (Research Gap) [완료]**: FastAPI BackgroundTasks 비동기 비교 분석 큐, 논문 한계점/해결과제 Pydantic 추출, LLM Synthesis를 통한 학술 공백 추론 및 추천 연구 주제 제안 개발.
*   **Case C: 보안 샌드박스 피어 리뷰 및 가설 디펜스 아레나 [완료]**: PDF 격리 업로드 및 임시 테이블 인덱싱, 3대 심사위원(방법론, 신규성, 문체) LangGraph 토론 프로세스 및 실시간 가설 방어 디펜스 아레나 개발.
*   **Case D: 연구 비서 Gem 팩토리 & 스토어 [완료]**: RAG 소스 필터 및 시스템 프롬프트 바인딩 커스텀 젬 생성, 젬 카드 그리드 및 1:1 대화방 컴포넌트 개발.
