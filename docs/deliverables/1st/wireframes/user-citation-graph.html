Evaluating Chunk Size in Academic Document Retrieval

J. Smith, K. Adams | 2024
MTEB Dataset SCIDOCS
Similarity 0.892 (Cosine)
Citations 42 times
Category Information Retrieval
Abstract

"We investigate the effect of text chunk size on semantic search accuracy within academic paper archives. In this work, we demonstrate that a 500-character chunk size with a 50-character overlap yields the optimal trade-off between semantic integrity and retrieval speed for dense retrievers, outperforming larger structural paragraphs."
