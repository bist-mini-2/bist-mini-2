# 📅 3인 전담제 기반 2주 단기 집중 개발 일정 및 WBS (Development Roadmap & WBS)

본 문서는 **6월 30일 프로젝트 완료 및 최종 배포**를 목표로, 3명의 팀원이 **1단계에서 3대 RAG 파이프라인을 공동 구축한 후, 2단계에서 기능 A, B, C, D 중 담당한 핵심 기능을 7일 동안 풀(Full)로 전담하여 구현**하는 평일 작업(총 11영업일) 기준 WBS 일정 계획서입니다.
